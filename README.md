# Heart Failure - Exploratory Data Analysis & Machine Learning

## Overview
This project performs Exploratory Data Analysis and Machine Learning 
on the Heart Failure Prediction dataset from Kaggle. The goal is to 
predict whether a patient has heart disease based on clinical features.

## Dataset
- Source: Kaggle - Heart Failure Prediction
- Samples: 918
- Features: 11 clinical features
- Target: HeartDisease (0 = No, 1 = Yes)

## Steps Covered
1. Basic Information - shape, columns, dtypes, statistical summary
2. Missing Values and Duplicates check
3. Decision Tree - plot and feature importance
4. Logistic Regression - classification report, confusion matrix, ROC curve
5. Naive Bayes - classification report, confusion matrix, ROC curve
6. Model Comparison

## Libraries Used
- pandas, numpy, matplotlib, seaborn, scikit-learn
